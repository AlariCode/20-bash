Задание 1 (/labs/09-01-awk/test_q1.sh)

1. В размещении `/home/moon/dataset.csv` находится набор данных с информацией о видеоиграх. Нам требуется проанализировать информацию из этой базы данных и выбрать оттуда названия игр.

2. У нас уже есть заготовка скрипта для извлечения названий `get_names.sh`. 
В данный момент этот сценарий проверяет, указано ли имя файла в качестве аргумента командной строки, существует ли этот файл и содержит ли какие-то данные.

Тебе необходимо дополнить этот скрипт под следующие требования:

- Определи новую переменную с названием `names_file` и присвой ей значение `/home/moon/names.txt`.
- Добавь строку кода, использующую `awk` для извлечения информации о названиях игр из каждой строки входного файла (В этом файле разделителями полей являются запятые, а нам интересно второе поле).
- Запиши извлеченную информацию в файл, указанный в переменной `names_file`.

Обрати внимание, что в выводе нам не требуется самая первая строка, которая является заголовком таблицы. Его можно удалить при помощи команды `tail -n +2`

Выполни скрипт, передав `/home/moon/dataset.csv` в качестве аргумента командной строки для проверки.
