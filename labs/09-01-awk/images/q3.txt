Одним из способов избежать вывода информации в каждой строке `awk` является использование блока `END`, который заставляет `awk` обрабатывать команду после завершения обработки входных данных.

Пример :
awk 'END {print "Hello"}' lines.txt

В дополнение к этому `awk` имеет две примечательные встроенные переменные, называемые

`NR` -> содержат количество строк из входной выборки.
`NF` -> содержит количество полей из входной образец.

Эти встроенные переменные обычно используются для доступа к информации об общем количестве строк, содержащихся в файле `(NR)`, 
и общем количестве столбцов, содержащихся в файле `(NF)`.


1. Обнови скрипт `/home/moon/file_analyzer.sh`, чтобы использовать `awk` с блоком `END` для подсчета общего количества строк и общего количества столбцов файла `/home/moon/dataset.csv`. 
Скрипт должен в выводе показывать только по одному значению для строк и для столбцов.
Также не забудь, что в файле нестандартный разделитель полей - `,`.
